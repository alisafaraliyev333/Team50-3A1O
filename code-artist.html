This is Code Art of Team 3A1O (Team 50) inspired by flag of Azerbaijan. This is an explanation of Code Art: Breaking down the code, it basicly creates a flag with three horizontal stripes. A white crescent and an 8-pointed star are positioned on the central red stripe. The drawCrescent function draws the crescent by finding the intercept points of the two circles and using those points to calculate the start and end angles for the outer and inner circle. The drawStar function creates an 8-pointed star by alternating between longer and shorter radius as it connects points around a circle.
